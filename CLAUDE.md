# Проект research-ux-jobs

## Граф знаний (graphify)
Готовый граф кода лежит в `.planning/graphs/graph.json` (850 nodes, 1372 edges). Он сохраняется между сессиями — пересобирать с нуля не нужно.

Перед тем как читать/грепать много файлов для вопросов об архитектуре, связях между файлами, "где живёт X", "что зависит от Y" — **сначала запрашивай граф**:
- через MCP-инструменты сервера `graphify` (подключён в `.mcp.json`), или
- `/gsd-graphify query <термин>`.

Читай конкретные файлы только после того, как граф указал, куда смотреть. Это экономит токены и не заставляет прыгать по всему репозиторию.

Пересборка графа нужна только после заметных изменений кода: `/gsd-graphify build`. Свежесть — `/gsd-graphify status`.
